# Sales-Excel-Dashboard
## Project Overview
This project analyzes a company's sales data using Microsoft Excel to uncover business insights. The dataset was cleaned, transformed, and analyzed using Pivot Tables and visualized through an interactive dashboard. 

## Business Problem 
The objective of this project is to analyze a company's sales data to identify key business insights. The analysis evaluates sales trends over the years, compares sales performance across states, examines monthly sales patterns, identifies the most profitable customers and products and assesses the performance of different product categories.

## Tools Used
Microsoft Excel
Excel Formulas
Pivot Tables
Pivot Charts
Slicers

## Dashboard Features
KPI Summary (Sales, Profit, Orders, and Customers)
Sales Trend by Year
Monthly Sales Trend
Sales by State
Sales & Profit by Category and Sub-Category
Top 5 Customers by Profit
Top 5 Products by Profit
